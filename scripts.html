標題主標卡
破除迷思 #03 ———————— 比鮮奶茶不甜 比拿鐵不苦 日式咖啡牛乳的三個標準
HOOK三杯對比
口播
「日式咖啡牛乳——」 「跟鮮奶茶不一樣，跟拿鐵也不一樣。」 「它有三個標準。」 「通不過的——」 「都不算日式咖啡牛乳。」
字幕
  • 不是鮮奶茶 / 不是拿鐵
  • 它有三個標準
STORY三個標準
口播｜標準一
「第一——不甜。」 「比鮮奶茶不甜。」 「糖只放夠『放大香氣』，不放更多。」
口播｜標準二
「第二——不苦。」 「比拿鐵不苦。」 「用萃取咖啡，不是 espresso。」
口播｜標準三
「第三——不澀。」 「奶要鮮乳、咖啡要好豆。」 「喝完——嘴裡是乾淨的。」
字幕
  • 標準一｜不甜（vs 鮮奶茶）
  • 糖 = 放大香氣，不是甜味
  • 標準二｜不苦（vs 拿鐵）
  • 萃取咖啡，不是 espresso
  • 標準三｜不澀
  • 鮮乳 + 好豆
  • 喝完嘴裡是乾淨的
OFFER系列承諾
口播
「不甜、不苦、不澀。」 「三條都通過——才是日式咖啡牛乳。」 （停一拍） 「下次買飲料——」 「拿這三條去檢查。」 「追蹤我繼續打破。」
字幕
  • 不甜｜不苦｜不澀
  • 三條都通過 = 日式咖啡牛乳
  • 下次買飲料 / 拿去檢查
  • 破除迷思 #03 + 大人牛乳
THREADS 配套貼文
